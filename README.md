# mantraCal

A basic calendar plugin for WordPress. At the moment 

## Current functions:
- Create, manage and delete events
- Event information: 
  - Title, 
  - Description, 
  - Event image, 
  - start date, 
  - end date,
  - start time, 
  - location
- Events can be output via shortcode:
  - Detailed view
  - monthly view
  - List view
